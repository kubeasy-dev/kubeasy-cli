    Create internal/deployer/cloud_provider_kind.go (new file, package deployer).

    Functions to implement:

    1. cloudProviderKindBinaryURL() string:
    ```go
    func cloudProviderKindBinaryURL() string {
        version := strings.TrimPrefix(CloudProviderKindVersion, "v")
        return fmt.Sprintf(
            "https://github.com/kubernetes-sigs/cloud-provider-kind/releases/download/%s/cloud-provider-kind_%s_%s_%s.tar.gz",
            CloudProviderKindVersion, version, runtime.GOOS, runtime.GOARCH,
        )
    }
    ```

    2. isCloudProviderKindRunning() bool:
    ```go
    func isCloudProviderKindRunning() bool {
        cmd := exec.Command("pgrep", "-f", "cloud-provider-kind")
        return cmd.Run() == nil
    }
    ```

    3. downloadCloudProviderKind(url, destPath string) error:
    - Create dest directory (os.MkdirAll) from filepath.Dir(destPath)
    - http.Get(url) the tar.gz
    - Read response body into memory (io.ReadAll)
    - Parse tar.gz: gzip.NewReader → tar.NewReader
    - Find entry where filepath.Base(hdr.Name) == "cloud-provider-kind" (exact binary name in archive)
    - Write to destPath via os.Create
    - Call os.Chmod(destPath, 0755)
    - Do NOT use kube.FetchManifest (that is for K8s YAML only and validates URLs against a K8s allowlist pattern)
    - Use net/http directly for binary download

    4. startCloudProviderKindDetached(binPath string) error:
    - Build exec.Command(binPath) with SysProcAttr{Setsid: true}, nil Stdout/Stderr/Stdin
    - Call cmd.Start() only, never cmd.Wait()

    5. ensureCloudProviderKind(ctx context.Context) ComponentResult:
    - If isCloudProviderKindRunning() → return ComponentResult{Name: "cloud-provider-kind", Status: StatusReady, Message: "already running"}
    - Download binary to GetCloudProviderKindBinPath() if not present (check os.Stat first)
    - If binary already downloaded (Stat succeeds): skip download
    - If binary not present: call downloadCloudProviderKind
    - Call startCloudProviderKindDetached
    - Return StatusReady on success; StatusNotReady on any error

Test the _ForPlatform variant.
