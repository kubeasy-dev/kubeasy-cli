<objective>
Thread ctx context.Context through all public functions in internal/api/client.go and update all cmd/ call sites to pass cmd.Context(), enabling Ctrl-C cancellation of in-flight HTTP requests.

Purpose: Currently, all API calls hardcode context.Background() — pressing Ctrl-C during a long `kubeasy challenge start` leaves the CLI hanging for 30 seconds until the HTTP timeout. After this change, Cobra's automatic Ctrl-C cancellation propagates to every HTTP request.

Output: Updated client.go with ctx param on 15 functions, 5 cmd/ files passing cmd.Context(), and all test anonymous functions updated to match new signatures.
</objective>
