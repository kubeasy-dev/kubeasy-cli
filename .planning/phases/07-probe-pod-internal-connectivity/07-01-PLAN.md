    CreateProbePod logic:
    1. Attempt DeleteProbePod with GracePeriodSeconds=0 to clear any stale pod (ignore error)
    2. Poll until stale pod disappears (wait.PollUntilContextTimeout, 500ms interval, 10s max, check IsNotFound on Get)
    3. Call clientset.CoreV1().Pods(namespace).Create() with the pod spec below
    4. Return (*corev1.Pod, error)

    Pod spec (per CONTEXT.md decisions):
    - Name: ProbePodName, Namespace: namespace
    - Labels: {"app": "kubeasy-probe", "managed-by": "kubeasy"}
    - RestartPolicy: corev1.RestartPolicyNever
    - One container: name="curl", image=probePodImage(), Command=["sleep","infinity"]
    - ImagePullPolicy: corev1.PullIfNotPresent
    - Resources.Requests: cpu=resource.MustParse("10m"), memory=resource.MustParse("16Mi")

    DeleteProbePod logic:
    - Call clientset.CoreV1().Pods(namespace).Delete(ctx, ProbePodName, metav1.DeleteOptions{GracePeriodSeconds: ptr(int64(0))})
    - If apierrors.IsNotFound(err): return nil
    - Otherwise return err
    - Add package-local ptr helper: func ptr[T any](v T) *T { return &v }

    WaitForProbePodReady logic:
    - wait.PollUntilContextTimeout(ctx, 1*time.Second, 30*time.Second, true, func(ctx context.Context) (bool, error) {
        pod, err := clientset.CoreV1().Pods(namespace).Get(ctx, ProbePodName, metav1.GetOptions{})
        if err != nil { return false, nil }
        return pod.Status.Phase == corev1.PodRunning, nil
      })
